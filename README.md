# Cawayan II Farmers Multi-Purpose Cooperative Website

An offline-capable website for the Cawayan II Farmers Multi-Purpose Cooperative, created as a local alternative to the Google Sites page.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Offline Capability**: All files are self-contained, no internet connection required
- **Search Functionality**: Quick search across officers, committees, and training
- **Interactive Elements**:
  - Click-to-copy for officer names
  - Dark mode toggle
  - Print-friendly layout
  - Smooth scrolling animations
  - Scroll-to-top button

## Content Organization

The website includes all information from the original Google Sites page:

- **Overview**: Business activity, products, and member statistics
- **Board of Directors**: Complete list of board members
- **Management Staff**: All management personnel
- **Committees**: Six committees with chairpersons and members
- **Training Attended**: List of completed training programs

## Usage

1. **Local Viewing**: Simply open `index.html` in any modern web browser
2. **No Server Required**: All functionality works without a web server
3. **Offline Ready**: Once downloaded, the website works completely offline

## Browser Compatibility

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers (iOS Safari, Chrome Mobile)

## Interactive Features

### Search
- Press `Ctrl+F` (or `Cmd+F` on Mac) to focus the search box
- Search across all content sections
- Highlighted results with smooth scrolling

### Dark Mode
- Click the moon icon in the bottom-left corner
- Preference is saved locally

### Print
- Click the print button in the bottom-right corner
- Optimized print layout with proper formatting

### Copy Names
- Click any officer name to copy it to clipboard
- Visual feedback when copied

## Technical Details

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with animations and transitions
- **Vanilla JavaScript**: No external dependencies
- **Responsive Grid Layout**: Adapts to all screen sizes
- **Local Storage**: Remembers dark mode preference
